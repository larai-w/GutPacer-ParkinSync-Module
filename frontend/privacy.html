GutPacer プライバシーポリシー

ベアイラボ

ベアイラボ(以下「運営者」)は、GutPacer(以下「本サービス」)における利用者の情報を次のとおり取り扱います。

取得する情報

  • LINEログインで取得する識別子(LINEユーザーID)、および表示名等の基本プロフィール。
  • 利用者が入力した排便・服薬(モビコール)・体調スコア・メモ・在宅/施設の別などの記録。
  • 通知の宛先となるLINEのユーザー識別子。

利用目的

  • 記録の保存・表示・PDF出力、およびLINEによるリマインド/警告通知の提供のため。
  • 不具合対応・品質改善のため。

管理者によるデータの閲覧

  • 運営者は、原則として利用者個々の記録内容を閲覧しません。
  • 障害調査・不具合対応の目的のときのみ閲覧することがあり、その場合は可能な限り事前に本人へ通知します。

第三者提供・委託

  • 本人の同意なく第三者へ提供しません。
  • データは Amazon Web Services(AWS)上に保存し、LINE通知は LINE Messaging API を利用します。これらはサービス提供に必要な範囲での委託先です。

保存期間・削除

  • 記録は本サービスの提供に必要な期間保存します。
  • 退会の申請があった場合、30日以内に当該利用者の記録を削除します。

開示・訂正・削除の請求

利用者は、自己の情報の確認・訂正・削除を請求できます。窓口は下記お問い合わせ先です。

安全管理

通信は暗号化し、認証を経たアクセスのみを許可します。データのバックアップを有効にしています。
