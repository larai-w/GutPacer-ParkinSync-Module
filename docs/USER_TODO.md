### H-4. LINE Login チャネル + LIFF アプリの作成
**⏱ 20〜30分 / 繰り返しなし**

1. LINE Developers コンソール → 既存プロバイダー → 「新規チャネル作成」→「LINE Login」
2. チャネル作成後、LIFF タブ → 「追加」
3. エンドポイント URL: `https://veai.jp/gutpacer/`、サイズ: Full
4. 発行された **LIFF ID**(例: `1234567890-AbCdEfGh`)と**チャネルID**を Claude に渡す → G-1 の実装に入れる

**つまずきポイント**: LIFF の「エンドポイントURL」はフロントのURLそのまま。変なパスは付けなくてよい。

### H-7. データ取り扱い方針を3点決める
**⏱ 15〜20分 / 繰り返しなし**

他人の健康情報を預かるにあたり、最低限の決定:

| 決める項目 | 選択肢の例 |
|---|---|
| ①あなた(管理者)はユーザーのデータを見るか | 「デバッグ時のみ、かつ本人に事前通知」など |
| ②退会時のデータ削除期限 | 「退会後30日以内に削除」など |
| ③問い合わせ対応者 | あなた本人のみ、など |

決まったら Claude に伝える → 規約の対応箇所に反映する。

### H-8. 料金方針を決める
**⏱ 10分 / 繰り返しなし**

| フェーズ | 推奨 | 理由 |
|---|---|---|
| 0 → 10人 | 無料 | フィードバック収集優先。課金で離脱リスクが出る |
| 10 → 30人 | 月¥300前後(任意) | LINE代¥5,000÷30人 = ¥167なので¥300で黒字 |
| 30人以降 | 要再検討 | 継続率データが出てから |

決まったら Claude に伝える → 利用規約の料金条項に書く。

**⏱ 初回リストアップ: 20分 / 連絡: 1人あたり10〜15分**

手順:
1. 訪問看護師・ケアマネ・神経内科関係で「在宅介護の家族と接点がある」人を2〜3人リストアップ
2. 頼む文言: 「パーキンソン病の在宅介護で排便記録に困っている家族がいたら紹介してほしい。無料のWebアプリを一緒に使ってみてもらいたい」
