# パーキンソン病の在宅介護で、排便記録を始めた理由

想定検索語: `パーキンソン病 便秘 記録` / `パーキンソン病 介護 記録`

パーキンソン病のある家族を在宅で介護していると、毎日の中にいくつもの確認があります。

薬は飲めたか。今日はいつもと違わないか。そして、排便はあったか。

その場では覚えていても、数日後に聞かれると答えに迷うことがあります。忙しかった日、施設に行っていた日、家族が交代で見ていた日。情報は少しずつ別々の場所に散らばっていきます。

GutPacerは、そんな経験から作り始めました。

## 「覚えておく」ことが、見えない仕事になる

介護の仕事には、手を動かすものだけではありません。

薬の時間を覚えておく。次の受診日を確認する。施設からの連絡を読む。排便があったかを気にかけておく。

どれも、誰かに頼まれて初めてする仕事ではないかもしれません。でも、頭の片隅から離れない。忘れてはいけないと思うほど、少しずつ疲れていきます。

その中でも排便のことは、記録しなければ後から確かめにくい情報です。だから、家族の記憶だけに預けない方法を作りたいと思いました。

## 最初に残したかった記録

GutPacerでは、排便の有無だけでなく、次のような情報を短く残せるようにしています。

- 量の目安
- 便の状態
- 浣腸や手助けの有無
- モビコールの服用状況
- その日のメモ
- 自宅にいるか、施設にいるか

これは、すべての家庭に同じ記録をしてほしいからではありません。パーキンソン病の在宅介護の中で、「あとから確認したい」と感じたものを、最初の形にしただけです。

家庭によって必要な項目は違います。将来は、薬や便の選択肢、通知のタイミングをそれぞれの家庭に合わせて設定できるようにしていく予定です。

## LINEに思い出してもらう

介護をしている人自身が、記録を忘れることもあります。忘れたことを責めたいわけではなく、忘れるほど別の仕事があったということです。

GutPacerでは、排便の記録がない日が続いたとき、LINEで記録を見直すきっかけを届けます。これは病気や状態を判断する通知ではありません。今日のことを確認する時間かもしれない、と気づくためのものです。

施設にいるときは、家族が自宅と同じようには記録できません。場所を切り替えられるようにしているのも、実際の介護の流れに合わせたかったからです。

## パーキンソン病から始めて、排便ケアへ

GutPacerはパーキンソン病の在宅介護から始まりました。でも、使う人をパーキンソン病の家族だけに限定したいわけではありません。

排便ケアを担っている家族には、病名が違っても、「今日のことを覚えておく」「あとで家族に伝える」「受診前に見返す」という似た仕事があります。

ひとつの家庭で感じた負担を、別の家庭でも使える小さな道具にする。その順番で、少しずつ作っていこうと思っています。

医療上の判断や治療の代わりではありません。気になる変化や服薬のことは、主治医や医療・介護の専門職にご相談ください。

GutPacerは、排便と服薬の記録を、家族の毎日に置ける形から始めています。
