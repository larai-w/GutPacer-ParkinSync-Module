# 排便記録が続かないのは、あなたのせいじゃない

想定検索語: `介護 排便記録 続かない` / `介護 記録 アプリ`

排便の記録、続いていますか。

最初はきちんと書ける。日付を並べて、便の状態もメモする。けれど、急な通院が入った日、眠れなかった翌日、仕事から帰って何もする気力が残っていない夜。記録は簡単に途切れます。

数日後、空白のページを見て、「また続かなかった」と思う。

でも、それは意志が弱いからではありません。介護の毎日が、記録帳の予定どおりには進まないだけです。

## 介護には、記録以外の仕事が多すぎる

排便のことを確認する。薬のことを覚えておく。施設の連絡帳を見る。次の受診の日を調べる。

一つひとつは小さく見えても、頭の中にはたくさんの仕事が並んでいます。記録を忘れた日は、何もしていない日ではありません。記録以外の仕事で一日が終わった日です。

だから、記録が空白になったことを、介護できなかった証拠にしないでください。

## 続く道具は、空白の日を責めない

記録を続けるために必要なのは、毎日完璧に書く意志より、戻ってきやすい仕組みかもしれません。

- 入力する項目を増やしすぎない
- 一度に短く記録できるようにする
- 空白の日があっても、最初からやり直しにしない
- 通知は責めるためではなく、思い出すきっかけにする

「昨日は書きませんでしたね」と言われるより、「今日の分だけ残しておきましょう」と言われる方が、もう一度開きやすい。私はそう思っています。

## 記録できた日だけを、大切にする

三日ぶりの記録も、一件の記録です。

そこには、その日の排便のことや、服薬のこと、家族が覚えていたことが残っています。毎日つながっていなくても、あとから見れば、いくつもの事実が並んでいる。それで十分役に立つことがあります。

記録は競争ではありません。連続日数を伸ばすためでも、誰かに褒めてもらうためでもない。未来の自分が「この頃はどうだったかな」と確認できるように、今日のことを一行だけ置いておくものです。

## 思い出す仕事を、少しだけ道具に渡す

GutPacerには、排便の記録がない日が続いたときに、LINEで思い出すきっかけを届ける仕組みがあります。通知は治療の判断をするものではなく、「記録を見直す時間かもしれません」と知らせるものです。

施設にいる期間は、家族がいつもどおり記録できないこともあります。そうした状況を残しておけるよう、自宅か施設かを記録する項目も用意しています。

それでも、書けない日はあります。書けない日は、そのままで大丈夫です。戻ってきた日に、今日の分だけ残せばいい。

医療上の判断や治療の代わりではありません。気になる変化や服薬のことは、主治医や医療・介護の専門職にご相談ください。

GutPacerは、記録を続けるためではなく、覚えておく仕事を少し軽くするための道具です。
